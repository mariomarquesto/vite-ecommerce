
# Vite Ecommerce

Este es un proyecto de ecommerce desarrollado con Vite y React.

## Instalación

1. **Clona el repositorio:**

   ```bash
   git clone https://github.com/mariomarquesto/vite-ecommerce.git)
   cd vite-ecommerce
Instala las dependencias:

Asegúrate de tener Node.js y npm instalados en tu máquina.
npm install

Scripts Disponibles
En el directorio del proyecto, puedes ejecutar los siguientes scripts:

npm run dev: Ejecuta la aplicación en modo de desarrollo con Vite.
npm run build: Compila la aplicación para producción.
npm run lint: Ejecuta ESLint para analizar el código y aplicar las reglas de linting.
npm run preview: Previsualiza la aplicación compilada en modo de producción.
Tecnologías Utilizadas
React: Biblioteca JavaScript para construir interfaces de usuario.
Vite: Herramienta de construcción rápida para aplicaciones web modernas.
Tailwind CSS: Framework CSS para diseño rápido y personalizable.
ESLint: Herramienta de análisis estático para identificar problemas en el código JavaScript y JSX.
PostCSS y Autoprefixer: Procesadores CSS para mejorar la compatibilidad del navegador.
Estructura del Proyecto
src/: Contiene los archivos fuente de la aplicación.
public/: Contiene archivos estáticos que se copian directamente durante la compilación.
